# 一个企业级官网实例
## 1.使用Vite进行项目构建
``` 
yarn create vite
```
## 2.基本结构
 - pages
   - index.jsx
 - widgets
   - Header.jsx
   - Hero.jsx
   - Solution.jsx
   - HowItWork.jsx
   - Clients.jsx
   - Contact.jsx
   - Footer.jsx
## 3.使用Tailwindcss构建结构
安装tailwindcss
```
yarn add -D tailwindcss postcss autoprefixer
```
生成tailwindcss.config.js和postcss.config.js文件
```
npx tailwindcss init -p
```
